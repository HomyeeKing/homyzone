## 什么是代理文件（Proxy Media）

代理文件（Proxy Media）是原始视频素材的低分辨率、低码率副本。在达芬奇（DaVinci Resolve）中，代理文件让你可以在性能较弱的电脑上流畅剪辑高分辨率素材（如 4K、6K、8K），而无需担心卡顿。

简单来说：**代理文件 = 原始素材的「轻量版替身」**

---

## 为什么要使用代理文件

### 1. 解决卡顿问题

| 原始素材 | 代理文件 |
|---------|---------|
| 4K/6K/8K 高分辨率 | 1080p 或更低 |
| 高码率（几百 Mbps） | 低码率（几 Mbps） |
| 电脑解码压力大 | 电脑轻松解码 |
| 时间线卡顿 | 时间线流畅 |

### 2. 实际场景

**场景一：笔记本剪辑 4K**
> 你的 MacBook Air 播放 4K H.265 素材卡顿，但生成代理后，剪辑如丝般顺滑。

**场景二：多机位剪辑**
> 4 机位 4K 同时播放，电脑直接卡死。使用代理后，多机位切换流畅自如。

**场景三：远程协作**
> 将代理文件发送给合作伙伴，无需传输几百 GB 的原始素材。

---

## 达芬奇的代理文件类型

达芬奇提供两种代理方式：

### 1. 优化媒体（Optimized Media）
- **特点**：达芬奇自动生成的代理文件
- **位置**：存储在项目文件夹中
- **适用**：个人工作，不跨设备

### 2. 代理媒体（Proxy Media）
- **特点**：用户手动生成的代理文件
- **位置**：可自定义存储位置
- **适用**：团队协作，跨设备共享

## 如何使用代理文件

### 第一步：生成代理文件

**方法 A：单片段生成**
1. 在媒体池（Media Pool）中选中素材
2. 右键 → **生成代理媒体**（Generate Proxy Media）
3. 等待生成完成

**方法 B：批量生成**
1. 媒体池中选择多个片段（Ctrl/Cmd + A 全选）
2. 右键 → **生成代理媒体**
3. 达芬奇会在后台批量处理

**方法 C：导入时自动生成**
1. 打开 **偏好设置** → **用户** → **剪辑**
2. 勾选 **自动将片段转为代理媒体**
3. 导入素材时自动创建代理

### 第二步：切换代理/原始模式

在剪辑页面，找到时间线预览窗口的右上角：

```
[原始] [代理] [自动]
```

| 模式 | 说明 |
|------|------|
| **原始** | 始终使用原始高分辨率素材 |
| **代理** | 始终使用低分辨率代理文件 |
| **自动** | 根据性能自动切换（推荐） |

**快捷键**：
- 切换代理模式：`Ctrl/Cmd + P`

### 第三步：交付时切换回原始

剪辑完成后，交付（Deliver）页面默认使用**原始素材**输出，无需手动切换。确保输出质量不会受代理文件影响。

## 代理文件设置建议

### 推荐参数

| 原始素材 | 代理分辨率 | 代理格式 | 码率 |
|---------|-----------|---------|------|
| 1080p | 720p | ProRes Proxy / H.264 | 10-20 Mbps |
| 4K | 1080p | ProRes Proxy / H.264 | 20-40 Mbps |
| 6K/8K | 1080p 或 720p | ProRes Proxy | 20-40 Mbps |

### 设置路径

**偏好设置** → **用户** → **剪辑** → **代理媒体**

可以设置：
- 代理格式（ProRes / H.264 / H.265）
- 代理分辨率（1/2、1/4、1/8）
- 代理存储位置

## 常见问题

### Q1：代理文件会占用多少空间？

代理文件通常只有原始素材的 **5%-15%**。例如：
- 原始 4K 素材：10 GB
- 代理文件：约 500 MB - 1.5 GB

### Q2：删除代理文件会影响原始素材吗？

**不会**。代理文件是独立的副本，删除后原始素材完好无损。你可以在媒体池中右键选择 **删除代理媒体** 来清理空间。

### Q3：为什么生成代理后还是很卡？

检查以下几点：
1. 是否切换到 **代理模式**（时间线右上角）
2. 代理分辨率是否还是太高（尝试 1/4 或 1/8）
3. 硬盘速度是否足够（代理文件建议放在 SSD）

### Q4：团队协作如何共享代理文件？

1. 统一代理文件存储路径（如项目文件夹内的 Proxy 子目录）
2. 将代理文件夹与项目文件一起打包传输
3. 接收方放置到相同路径，达芬奇会自动识别

## 最佳实践

1. **导入即生成**：设置自动转代理，省去手动操作
2. **合理分辨率**：不需要太极端，1080p 代理足够剪辑 8K
3. **SSD 存储**：代理文件放在 SSD 上，播放更流畅
4. **定期清理**：项目完成后删除代理文件，释放空间
5. **原始备份**：代理文件不能替代原始素材，务必做好备份

## 总结

代理文件是达芬奇剪辑工作流程中的重要工具，它让你可以在普通电脑上流畅处理高分辨率素材。记住核心要点：

> **剪辑用代理，交付用原始**

掌握代理文件的使用，你的剪辑效率将大幅提升！
