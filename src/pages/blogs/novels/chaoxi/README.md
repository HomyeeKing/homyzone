## 故事简介

二十年前的一场阴谋，让一个显赫家族轰然倒塌。
二十年后，一个普通的城市青年，意外揭开了历史的真相。

是命运的捉弄，还是宿命的救赎？
当潮汐退去，真相浮现，
他能否在风暴中找到自己的方向？
